Summary

MS Graph API is used to retrieve the information of the events going on objects that have been shared using Microsoft tools, such as Microsoft Outlook, Microsoft Teams, etc. The data retrieved will be used for Organizational Network Analysis (ONA). It comprises of algorithms, which measure the efficiency of the teams in the organization and employee engagement.

The algorithms must be able to smoothly access the data. Hence, the data is stored in databases, to help the developer write clean code to access the data. Data collected may include the following.

  • Details of all the users in the organization
  • Mails sent across users
  • Calendar events of users
  • Details of conversations and chats which have taken place between users
  • To do list of users

Tools used

  • MS Graph API
  • Bottle Server
  • SQLite3 Database
